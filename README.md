# Sistema Verificador de Números

Este projeto é um **sistema simples de verificação de números**, desenvolvido com o objetivo de **praticar lógica de programação**, estruturas condicionais e entrada/saída de dados.

O sistema analisa um número informado pelo usuário e retorna informações relevantes sobre ele.

---

## Funcionalidades

- Verificar se o número é **par ou ímpar**
- Verificar se o número é **positivo, negativo ou zero**
- Exibir o resultado de forma clara no terminal
- Interface simples e intuitiva

---

## Conceitos Utilizados

- Variáveis
- Estruturas condicionais (`if`, `else`)
- Operadores matemáticos
- Entrada e saída de dados
- Lógica de programação

## Tecnologias Utilizadas

- Linguagem: **JavaScript** **HTML/CSS**
